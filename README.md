# Forschungsportal thefreshmind4o

Portal des Forschungsökosystems: sichtbare Projektstände, Verweise und DOI-Pfade.

## Rolle im Forschungsökosystem
Dieses Repository liefert Portal mit Projektübersicht und DOI-Pfaden. Es ist Teil eines fünfteiligen Ökosystems (siehe [Architekturmodell](./ARCHITEKTURMODELL.md)):

1. `forschungsdaten-und-schemata`
2. `psycho-semantischer-guardrail`
3. `x-matrix-kern`
4. `modellbibliothek-identitaet`
5. `thefreshmind4o.github.io`
